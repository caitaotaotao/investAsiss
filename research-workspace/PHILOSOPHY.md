# 投资哲学约束

> 本文件为核心投资哲学约束文档，仅支持人工确认后更新。反思模块可提出修改建议，但需人工确认方可生效。

---

## 一、核心理念

### 1.1 周期是研究的底层框架

相信周期为研究的本质。市场周期、行业周期、公司生命周期构成投资研究的底层框架。任何公司都处于周期的某一阶段——**复苏、成长、成熟、衰退**。研究的本质在于：

- 识别公司当前所处的周期位置
- 预判周期转折点的时间与触发条件
- 把握趋势的延续与逆转信号

### 1.2 追溯基本面变化的深层驱动因素

财务数据是结果，而非原因。理解一家公司，需持续追问：

- 是什么决定了它的竞争优势？
- 这种优势能持续多久？
- 什么因素会导致其基本面变好或变差？
- 这种驱动因素是逐渐加强，还是很快会扩散至竞争对手？

## 二、投资逻辑分类

### 2.1 红利价值股

**核心逻辑**：寻找被低估的稳定现金流资产，以估值修复（PE回归）和分红收益为主要收益来源。

**适用特征**：
- 盈利稳定、现金流充沛、派息率可持续
- 行业成熟、竞争格局稳定、颠覆性变化概率低
- 分红收益率具备相对无风险利率的吸引力
- 典型行业：公用事业、金融、传统消费龙头（白色家电、白酒）

**核心研究重点**：

| 维度     | 关注要点                                                         |
|--------|--------------------------------------------------------------|
| 分红可持续性 | 分红政策稳定性；国央企需关注管理层对分红的政策表态及公告承诺                               |
| 资产负债表  | 偿债能力健康度；是否存在短期内影响自由现金流的重大资本开支计划                              |
| 竞争格局   | 行业格局是否稳固；警惕特许经营权政策变化、行业整体快速衰败风险                              |
| 煤炭股特殊性 | 动力煤价格变化趋势；煤价上涨叠加红利风格轮动可带来显著超额收益                              |
| 银行股特殊性 | 非息差业务成长性（一线城市城商行、股份行优势业务）；需综合考量宏观周期 + 防御属性 + 指数权重（国家队护盘）三重特征 |

---

### 2.2 成长股

**核心逻辑**：寻找第二增长曲线；把握产业链核心环节的国产替代机会，捕捉中国产业链整体向高附加值演进的结构性红利。

**适用特征**：
- 新技术、新客户、新市场驱动的非线性增长
- 行业处于渗透率快速提升阶段或竞争格局重塑窗口期
- 增长逻辑兑现带来业绩超预期，为核心收益来源
- 典型行业：科技、创新药、新能源、高端制造

**核心研究重点**：

| 维度     | 关注要点                     |
|--------|--------------------------|
| 增长驱动力  | 技术壁垒、客户粘性、网络效应的可持续性      |
| 渗透率曲线  | 当前所处位置（早期快速渗透 vs 晚期增速减缓） |
| 竞争壁垒   | 壁垒强度与可持续时长的动态评估          |
| 业务验证节点 | 新业务/新客户的扩张节奏与关键里程碑       |
| 戴维斯双击  | 识别业绩兑现与估值扩张共振的时间窗口       |

---

### 2.3 周期股

**核心逻辑**：把握周期上行启动窗口，利用股价领先基本面的时间差获取超额收益。周期股的核心难点不在于"买什么"，而在于"什么时候关注、什么时候退出"。

**适用特征**：
- 盈利与行业/宏观周期高度相关，业绩波动大
- 行业供需格局是核心驱动变量（产能周期、库存周期）
- 典型行业：化工、有色金属、钢铁、航运、养殖、半导体（设备周期）

**核心研究重点**：

| 维度 | 关注要点 |
|------|---------|
| 周期位置判断 | 当前处于去库存末期/补库存初期？产能出清程度？行业开工率/库存水平等先行指标 |
| 股价与基本面的时间差 | **股价往往领先基本面 1-2 个季度见顶**；当基本面最好时（盈利高点），股价可能已在下跌 |
| 上行启动信号 | 行业底部特征：产能出清充分、库存降至历史低位、龙头率先提价成功、下游需求边际改善 |
| 顶部预警信号 | 产能大规模扩张、库存快速累积、行业利润率创历史新高（均值回归压力）、市场一致看多 |
| 估值陷阱 | 周期股高 PE 往往出现在盈利低谷（分母小），低 PE 反而出现在盈利高点（分母大）；不应以静态 PE 高低作为买卖依据 |

**与成长股的关键区别**：
- 成长股的增长是结构性的（渗透率提升），可长期持有
- 周期股的增长是波动性的（供需摆动），必须择时退出
- 误将周期复苏当作趋势成长，是最常见的亏损模式之一（见 §4 周期误判风险）

## 三、研究方法论

### 3.1 数据与基本面趋势验证原则

1. **先假设，后验证**
  在阅读财报或研报之前，先独立形成假设（增长点、风险点），再用数据验证，避免被既有叙事锚定。

2. **趋势比点位重要**
  关注变化的方向与斜率，而非绝对值的静态截面分析。

3. **多重交叉验证**
  单一数据源存在失真风险，需从多个维度交叉印证：
  - 财务数据（收入、利润、现金流）
  - 经营数据（出货量、订单、产能利用率）
  - 行业数据（渗透率、竞争格局、政策动向）

4. **警惕预期差**
  市场共识与实际结果之间的偏差，是真正的交易机会所在。

---

### 3.2 增长曲线机会识别框架

**五类结构性增长机会**：

| 类型   | 描述                 | 代表方向       |
|------|--------------------|------------|
| 新技术  | 技术变革带来的结构性机会       | AI、云计算、新能源 |
| 新客户  | B端/C端新客群开拓，验证产品普适性 | 消费品出海、企业服务 |
| 新市场  | 国际化扩张、新兴市场渗透       | 东南亚、中东市场   |
| 第二曲线 | 原有业务见顶前，新业务已验证放量   | 平台型公司转型    |
| 国产替代 | 高端领域国内企业份额提升       | 半导体、创新药    |

**识别"真增长"与"伪增长"**：

| 判断维度 | 真增长 ✅      | 伪增长 ❌         |
|------|------------|---------------|
| 客户行为 | 主动选择、持续复购  | 靠折扣/激励拉动，流失率高 |
| 盈利质量 | 毛利率稳定或持续提升 | 增收不增利，利润率下滑   |
| 增长来源 | 真实需求驱动     | 渠道压货、会计处理优化   |

## 四、风险意识框架

> 以下为研究过程中需持续保持警觉的系统性风险维度：

- **估值风险**：成长股高估值对增长兑现的高度依赖，一旦增长低于预期，杀估值与杀业绩双重打击
- **周期误判风险**：将周期性复苏误判为趋势性成长，在基本面最好（但股价已领先见顶）时高位接盘；反向亦然——在基本面最差时恐慌卖出，错过周期反转（参见 §2.3）
- **叙事陷阱**：宏大叙事掩盖基本面恶化，需以数据验证替代故事驱动
- **流动性风险**：小市值、低流动性标的在市场风格切换时的极端波动
- **政策风险**：监管政策变化对特定行业（互联网、教育、医疗）的非线性冲击
